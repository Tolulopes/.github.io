General Assembly

The Web Development Immersive is a 12 week programme which covers, Ruby, Ruby-on-rails, Javascript, Jquery etc. During this time I completed a number of labs, three projects, and one hackathon.

Languages

The course mostly covers ruby, but we also learnt Javascript, Jquery, HTML,CSS, AngularJS

The Future

As the course only covered a tiny bit of Angular I would really like to be able to build an entire app/ site using angular I am currently doing the tutorials on code school and hoping to create a site using angular.

Projects that I completed

Holiday Transfer

This was my final project where I deicded to create a Holiday Transfer site which is essentially a market place for people sell their entire holiday package, so this includes flights, hotels, and any pre-booked attractions. I built this site using rails, javascript, jquery, stripe, devise, and gmaps4 rails.

Get Fit

This App is for clients to sign up and register and search for different classes that are held. Instructors are able to accept or reject requests, and once clients have attended the class they are able to leave comments on the class/instructor. This was built in rails using javascript, jquery, devise, and ransack.

Sound Board

This was a really cool lab that I enjoyed doing its a sounboard which plays different "beyonce" sounds "on click" events. This was built using javascript and soundmanager

HighclassHooliganHipster

This was a 24hour hackathan where myself and one other student built a really fun app based on three London Sub-cultures Highclass, Hooligan, and Hipster which would create an Intenerary for a three day stay in London. The twist was that the itineraries linked to each sub-cultures would be mixed up so you would have a hipster attending a highclass event and so on.
